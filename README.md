# 💰 Expense Tracker

A Full Stack Expense Tracker built using React.js, Django REST Framework, JWT Authentication and Chart.js.

## 🚀 Features

- User Authentication (JWT)
- Register & Login
- Add Expense
- Edit Expense
- Delete Expense
- Search Expense
- Filter by Category
- Filter by Date
- Income & Expense Dashboard
- Pie Chart
- Bar Chart
- Responsive Design
- PDF Export

## 🛠 Tech Stack

Frontend
- React.js
- Bootstrap
- Axios
- Chart.js

Backend
- Django
- Django REST Framework
- JWT Authentication

Database
- SQLite
